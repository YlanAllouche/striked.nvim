# Fixture Overview

This file documents the purpose of the markdown fixtures:

- `tasks.md`: scanner coverage for all known task states
- `reporting.md`: log and focused item coverage
- `rich_copy.md`: rich markdown rendering, metadata badges, and headings
- `browser_bookmarks.md`: browser tab insertion target and bookmark examples
- `journal/`: dated journal pages for prompt, today, next, previous, and picker tests
- `meetings/`: frontmatter-backed meeting notes for Telescope meeting picker tests

No task-like lines below this paragraph means the file should not affect picker results.
